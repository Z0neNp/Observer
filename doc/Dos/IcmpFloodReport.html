    class Dos::IcmpFloodReport

    A class for DoS interpretation data for ICMP protocol. “data” had to be removed from class name, due to restrictions on class name length. We have additional “DosIcmp” in class name, because Rails translates class name into table name and neglects namespacing before class name.
